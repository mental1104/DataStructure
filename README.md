# DSA

本仓库主要涵盖《数据结构-邓俊辉》和 *Algorithm 4th* 中的代码:

使用方法： 
测试操作系统：macOS/Ubuntu 22.04    
测试编译器：g++/clang  
需要安装cmake  

操作方式：`sh entry.sh`  
构建出来的二进制文件在build文件夹下。
